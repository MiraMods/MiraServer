# MiraServer
## What is this?
MiraServer is an [Impostor v1.2.2](https://github.com/MiraMods/Impostor) server hosted by Mira that allow players to play Among Us v2020.11.17/2020.12.9 in 2026. This server supports mods. It also supports more than 10 players in a lobby. Join our [Discord](https://discord.gg/hAKrVn5K9F) for more info.

#### Manual Install
1. Open the Steam console by pressing Win+R and run the following command.
```
steam://open/console
```
2. Run the following command in Steam console.
```
download_depot 945360 945361 3306639722673334636
```
3. Open the path `C:\Program Files (x86)\Steam\steamapps\content\app_945360\depot_945361` in Explorer.
4. Move the files in the folder to a new folder of your choice. (Optional)
5. Create a file named `steam_appid.txt` in the new folder.
6. Open the file in Notepad and type in `945360` and save the file.
7. Download [regioninfo.dat](https://raw.githubusercontent.com/MiraMods/MiraServer/refs/heads/main/regionInfo.dat).
8. Press Win+R and type
```
AppData\LocalLow\Innersloth\Among Us
```
9. Move the file you downloaded into the folder that opens.
11. Launch Among Us.
